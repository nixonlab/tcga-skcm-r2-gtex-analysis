# GTEx Analysis

## Sources

### `1.count_data.RData` 

Sent from Phoebe from original analysis

### `GTEx_Analysis_2025-08-22_v11_RNASeQCv2.4.3_gene_reads.parquet`

Contains the count matrix for GTEx. Public accessible download.
From [GTEx website](https://gtexportal.org/home/downloads/adult-gtex/bulk_tissue_expression)

### `meta.gtex.rds`

GTEx sample metadata (sample attributes)

Created by script if not present, download from [GTEx google storage](https://storage.googleapis.com/adult-gtex/annotations/v11/metadata-files/GTEx_Analysis_v11_Annotations_SampleAttributesDS.txt)

### GTEx color palette

Loaded by script from [Broad Institute GitHub](https://github.com/broadinstitute/gtex-viz/raw/f9d18299b2bcb69e4a919a4afa359c99a33fbc3b/boxplot/dev/colors.json)
